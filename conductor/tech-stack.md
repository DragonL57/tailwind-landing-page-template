# Tech Stack

## Frontend
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **UI:** React 19
- **Styling:** TailwindCSS v4

## Key Dependencies
- **Animations:** Framer Motion, AOS
- **Icons:** Lucide React
- **State:** HeadlessUI React
- **AI/ML:** Microsoft Azure Speech SDK

## Architecture
- Next.js App Router
- Route groups for multi-product support
- Component-based UI library

## Project Structure
```
app/            # Next.js App Router
components/      # React components
lib/            # Utilities and helpers
```
